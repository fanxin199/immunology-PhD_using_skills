# pub-grade-barchart

## Overview / 概览

`pub-grade-barchart` focuses on clean, publication-quality bar charts for biological and bioinformatics data.

`pub-grade-barchart` 适合快速生成发表级柱状图，兼顾 R 与 Python 生态，便于在不同项目里复用相同的图形标准。

## Included / 包含内容

- `SKILL.md` as the main guide
- `scripts/` and `examples/` for both Python and R patterns
- `references/api_reference.md` for usage details
- `assets/` for lightweight bundled resources

## Best For / 适用场景

- Figure panels with grouped comparisons
- Supplementary plots that still need consistent styling
- Teams that move between R and Python plotting stacks

## Notes / 说明

This skill is narrower than `bioviz-pro`, but it is faster to reach a clean bar-chart output when that is the only plotting need.
