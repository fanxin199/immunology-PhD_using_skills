# enhanced-volcano

## Overview / 概览

`enhanced-volcano` is a focused volcano-plot skill built around the R package `EnhancedVolcano`.

`enhanced-volcano` 专注于差异表达火山图，适合需要自动避让标签、突出特定基因集、并输出发表级矢量图的场景。

## Included / 包含内容

- `SKILL.md` with plotting guidance and parameter notes
- `scripts/example.R` as a minimal working pattern
- `references/parameters.md` for styling and labeling choices

## Best For / 适用场景

- DEG result visualization for papers and presentations
- Highlighting named genes or pathways in a dense volcano plot
- Producing a polished result without building the plot layer-by-layer from zero

## Notes / 说明

Use this when the task is specifically a volcano plot; use `bioviz-pro` when you need a broader plotting toolkit.
