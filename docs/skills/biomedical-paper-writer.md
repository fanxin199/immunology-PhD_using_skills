# biomedical-paper-writer

## Overview / 概览

`biomedical-paper-writer` is a writing-centered skill for biomedical manuscripts, from planning the paper story to polishing sections for journal submission.

`biomedical-paper-writer` 面向生物医学论文写作与修改，覆盖从论文结构设计、章节撰写到投稿前语言与图表规范检查的完整流程。

## Included / 包含内容

- `SKILL.md` with a detailed writing workflow
- `references/imrad_guide.md` for section-by-section drafting
- `references/language_checklist.md` for style and grammar review
- `references/figures_tables.md` for figure and table conventions

## Best For / 适用场景

- Drafting original research papers and review articles
- Revising Introduction, Methods, Results, Discussion, and Abstract
- Aligning scientific writing with a clearer story structure before submission

## Notes / 说明

This skill is Chinese-first in writing guidance but explicitly supports English biomedical manuscript work as well.
