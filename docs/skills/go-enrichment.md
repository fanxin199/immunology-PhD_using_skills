# go-enrichment

## Overview / 概览

`go-enrichment` is a compact interpretation skill for Gene Ontology enrichment in immunology and transcriptomics workflows.

`go-enrichment` 是一个轻量但实用的 GO 富集解释技能，覆盖 ORA 与 ranked/GSEA 风格分析，重点强调背景集、ID 映射和多重检验这些最容易做错的地方。

## Included / 包含内容

- `SKILL.md` with practical guidance
- Example code patterns for Python-oriented enrichment workflows

## Best For / 适用场景

- Interpreting DEG lists or ranked gene lists
- Immunology-themed biological process summaries
- Teaching or standardizing enrichment analysis decisions across projects

## Notes / 说明

This skill is intentionally small: it acts more like a decision guide and workflow reference than a script bundle.
