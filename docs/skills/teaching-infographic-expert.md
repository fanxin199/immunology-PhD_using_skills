# teaching-infographic-expert

## Overview / 概览

`teaching-infographic-expert` converts dense textbook or chapter content into English infographic blueprints optimized for AI image-generation workflows.

`teaching-infographic-expert` 用于把教材章节或复杂医学文本转成适合教学信息图生成的英文蓝图，强调结构清晰、模块完整和视觉表达友好。

## Included / 包含内容

- `SKILL.md` with blueprint and image-generation workflow
- `references/example_chapter.md` as a source-text example
- `references/example_guidance.md` as a target-output example

## Best For / 适用场景

- Turning textbook chapters into infographic planning documents
- Preparing visual teaching material for immunology or medical education
- Converting dense narrative text into modular, drawable visual instructions

## Notes / 说明

This skill requires English-only blueprint output and targets a 16:9 infographic format for teaching use.
