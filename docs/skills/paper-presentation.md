# paper-presentation

## Overview / 概览

`paper-presentation` turns academic paper PDFs into talk-ready presentation materials and study packs.

`paper-presentation` 用于把论文 PDF 转成可汇报的讲稿材料，既可以生成中文 PPT 工作流，也可以生成适合自学和讨论的论文学习包。

## Included / 包含内容

- `SKILL.md` with branching logic for slides vs study sidecar
- `scripts/` for figure extraction, text extraction, montage generation, rendering, and cleanup
- `references/` for research paper, review paper, and visual-review handling
- `agents/openai.yaml` metadata for interface-facing descriptions

## Best For / 适用场景

- Journal club and lab meeting slide preparation
- Reading-heavy paper study workflows
- Figure-centric paper interpretation when the user wants both structure and visuals

## Notes / 说明

This skill already contains reusable tooling and metadata, so the repository keeps it close to its working structure.
