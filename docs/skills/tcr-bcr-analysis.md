# tcr-bcr-analysis

## Overview / 概览

`tcr-bcr-analysis` covers immune repertoire analysis for TCR and BCR clonotypes, especially when V(D)J data must be integrated with single-cell RNA workflows.

`tcr-bcr-analysis` 面向 TCR/BCR 克隆型分析，强调克隆扩增、克隆多样性、样本重叠度以及与 AnnData/scRNA 数据的整合。

## Included / 包含内容

- `SKILL.md` with workflow guidance and metric definitions
- Detailed notes for 10x V(D)J inputs, barcode normalization, overlap, diversity, and AnnData integration

## Best For / 适用场景

- 10x V(D)J clonotype analysis
- Linking clonotype states back to cell clusters or UMAP coordinates
- Repertoire summaries in immunology projects and papers

## Notes / 说明

This skill is text-heavy by design and acts as a practical workflow reference rather than a script bundle.
