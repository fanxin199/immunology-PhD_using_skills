# squidpy-spatial-viz

## Overview / 概览

`squidpy-spatial-viz` is a focused visualization skill for spatial transcriptomics data, centered on the plotting capabilities of Squidpy.

`squidpy-spatial-viz` 专注于空间转录组可视化，适合在已有空间数据对象的基础上快速生成空间散点图、组织切片叠加图和空间特征展示图。

## Included / 包含内容

- `SKILL.md` with plotting guidance
- `scripts/` for neighborhood analysis, spatial feature plotting, and scatter visualizations
- `references/` and `assets/` for reusable plotting context

## Best For / 适用场景

- Publication-ready spatial figures
- Tissue overlays and feature maps
- A lightweight visualization companion to the broader `spatial-transcriptomics` workflow

## Notes / 说明

Use this skill when the main need is figure output rather than the full spatial analysis pipeline.
