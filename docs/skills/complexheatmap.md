# complexheatmap

## Overview / 概览

`complexheatmap` packages an R-first workflow for rich, layered heatmaps using the `ComplexHeatmap` ecosystem.

`complexheatmap` 针对高级热图场景，适合把表达矩阵、分组注释、临床信息和多来源结果放进同一张高质量复合热图。

## Included / 包含内容

- `SKILL.md` as the main guide
- `scripts/template_generator.R` for bootstrapping figure code
- `references/` covering basic usage, annotations, advanced features, and gallery-style patterns
- `examples/` with ready-to-run R scripts

## Best For / 适用场景

- Genomics and transcriptomics heatmaps with many annotations
- OncoPrint or multi-panel summary heatmaps
- Figure generation when default `pheatmap` is no longer enough

## Notes / 说明

This is the most reference-heavy skill in the repo and is intentionally kept as a reusable R heatmap toolkit.
