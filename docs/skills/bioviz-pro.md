# bioviz-pro

## Overview / 概览

`bioviz-pro` is a publication-grade visualization skill for bioinformatics figures. It centers on `ggplot2` plus `ggsci` journal palettes and provides ready-to-use R scripts for common figures.

`bioviz-pro` 面向生物信息学发表级图形制作，核心依赖 `ggplot2` 和 `ggsci` 期刊配色，适合快速做出风格统一、可直接用于论文和汇报的图。

## Included / 包含内容

- `SKILL.md` with workflow guidance
- `scripts/` for volcano, MA, heatmap, PCA, survival, boxplot, correlation, enrichment, and Manhattan plots
- `references/color_palettes.md` for journal-style palette choices

## Best For / 适用场景

- Bulk RNA-seq or scRNA-seq differential expression figures
- Figure polishing for manuscripts, thesis chapters, and slides
- Reusing high-end journal color systems without redesigning plot themes from scratch

## Notes / 说明

This skill is R-heavy and works best when you already have tidy data frames ready for plotting.
