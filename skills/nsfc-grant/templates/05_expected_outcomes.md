# 预期成果 Prompt 模板

## 系统提示词

```
你正在撰写国自然申请书的「预期成果」部分。

### 核心要求

1. **学术成果**
   - 论文数量和级别要具体
   - 与项目规模匹配
   - 体现研究产出

2. **知识产权**
   - 专利、软著等（如适用）
   - 根据研究性质决定

3. **人才培养**
   - 研究生培养计划
   - 体现项目的育人功能

4. **应用价值**
   - 说明技术应用前景
   - 体现研究的社会价值

### 格式要求

- 分类列出各类成果
- 指标具体、可量化
- 总字数建议：300-500字
```

---

## 用户提示词模板

```
研究主题：{research_topic}

项目类型：{project_type}（如：面上项目、青年项目）

项目周期：{duration}（如：3年）

请撰写「预期成果」部分。
```

---

## 标准结构模板

```markdown
## 预期成果

### 学术论文

- 在国际高水平期刊（IF>10）发表SCI论文 **2-3** 篇
- 在本领域主流期刊发表SCI论文 **3-5** 篇

### 知识产权

- 申请国家发明专利 **1-2** 项

### 人才培养

- 培养博士研究生 **1-2** 名
- 培养硕士研究生 **2-3** 名

### 学术交流

- 在国际/国内学术会议上作报告 **2-3** 次

### 应用价值

[描述研究成果的潜在应用前景，1-2句话]
```

## 不同项目类型参考标准

### 面上项目（4年，~60万）

| 成果类型 | 建议数量 |
|----------|----------|
| SCI论文（高水平） | 2-3篇 |
| SCI论文（主流期刊） | 3-5篇 |
| 专利 | 1-2项 |
| 博士生 | 1-2名 |
| 硕士生 | 2-3名 |

### 青年项目（3年，~30万）

| 成果类型 | 建议数量 |
|----------|----------|
| SCI论文（高水平） | 1-2篇 |
| SCI论文（主流期刊） | 2-3篇 |
| 专利 | 0-1项 |
| 硕士生 | 1-2名 |

### 重点项目（5年，~300万）

| 成果类型 | 建议数量 |
|----------|----------|
| SCI论文（高水平） | 5-8篇 |
| SCI论文（总计） | 15-20篇 |
| 专利 | 3-5项 |
| 博士生 | 3-5名 |
| 硕士生 | 5-8名 |

## 常见问题与改进建议

| 问题 | 改进建议 |
|------|----------|
| 论文数量过多 | 与项目经费和周期匹配 |
| 指标过于保守 | 适当提高预期，体现雄心 |
| 缺少知识产权 | 根据研究性质添加专利 |
| 人才培养空白 | 补充研究生培养计划 |
| 应用价值缺失 | 简述潜在转化前景 |
