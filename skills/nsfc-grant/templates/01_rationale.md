# 立项依据 Prompt 模板

## 系统提示词

```
你正在撰写国自然申请书的「立项依据」部分。

### 核心要求

1. **研究背景与意义**
   - 阐明研究的科学意义和应用价值
   - 从宏观到微观层层递进
   - 突出研究的重要性和紧迫性

2. **国内外研究现状**
   - 客观评述国内外研究进展
   - 分主题或时间线组织
   - 明确指出存在的问题和不足

3. **研究缺口**
   - 明确指出现有研究的不足
   - 说明尚未解决的关键科学问题
   - 论证研究空间的存在

4. **本研究必要性**
   - 论证开展本研究的必要性和可行性
   - 引出核心科学假说
   - 自然过渡到研究目标

### 格式要求

- 文献引用使用 [1]、[2] 格式标注
- 段落清晰，逻辑连贯
- 总字数建议：2000-3000字

### 写作风格

- 专业、严谨、客观
- 避免过度夸大或贬低
- 用词准确，术语规范
```

---

## 用户提示词模板

```
研究主题：{research_topic}

补充信息：{additional_info}

请撰写「立项依据」部分。
```

## 免疫学领域示例结构

### 肿瘤免疫方向

1. **背景**：肿瘤免疫治疗的兴起与挑战
2. **现状**：
   - 免疫检查点抑制剂的成功与局限
   - 肿瘤微环境（TME）的研究进展
   - 免疫细胞亚群的功能研究
3. **缺口**：[具体科学问题]
4. **必要性**：本研究的切入点和意义

### 炎症免疫方向

1. **背景**：炎症与疾病的关联
2. **现状**：
   - 炎症小体/细胞焦亡/代谢重编程研究进展
   - 天然免疫与适应性免疫的交互
   - 炎症调控的分子机制
3. **缺口**：[具体科学问题]
4. **必要性**：本研究的创新切入点

## 常见问题与改进建议

| 问题 | 改进建议 |
|------|----------|
| 背景太宏大，与本研究关联弱 | 聚焦到具体科学问题，逐步收窄 |
| 现状罗列过多，缺乏评述 | 加入分析和评价，指出问题 |
| 研究缺口不明确 | 用「然而」「但是」明确转折 |
| 科学问题不够具体 | 凝练成一句话的核心科学问题 |
| 文献引用格式不统一 | 统一使用 [1]、[2] 格式 |
