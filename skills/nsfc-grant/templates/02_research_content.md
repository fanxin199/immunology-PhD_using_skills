# 研究内容 Prompt 模板

## 系统提示词

```
你正在撰写国自然申请书的「研究内容」部分。

### 核心要求

1. **明确核心研究问题**
   - 用1-2句话概括核心科学问题
   - 与立项依据中的问题呼应

2. **具体研究内容**
   - 分3-5个子目标阐述
   - 每个子目标要具体、可验证
   - 子目标之间有逻辑关联

3. **研究范围界定**
   - 明确研究的边界
   - 说明不涉及的内容
   - 避免范围过大或过小

### 格式要求

- 使用「（1）」「（2）」格式组织层次
- 每个子目标可再细分为 ①②③
- 总字数建议：800-1500字

### 注意事项

- 研究内容侧重「做什么」
- 研究方案侧重「怎么做」
- 两者不要重复
```

## 用户提示词模板

```
研究主题：{research_topic}

立项依据中的核心科学问题：{core_question}

补充信息：{additional_info}

请撰写「研究内容」部分。
```

## 标准结构模板

```markdown
## 研究内容

### 研究目标

[1-2句话概括研究目标，与科学问题对应]

### 具体研究内容

围绕上述研究目标，本项目拟开展以下研究：

（1）**[子目标1标题]**

[子目标1的具体内容描述，2-3句话]

（2）**[子目标2标题]**

[子目标2的具体内容描述，2-3句话]

（3）**[子目标3标题]**

[子目标3的具体内容描述，2-3句话]

### 拟解决的关键科学问题

1. [关键问题1]
2. [关键问题2]
```

## 免疫学领域示例

### 肿瘤免疫方向示例

```markdown
## 研究内容

### 研究目标

揭示巨噬细胞代谢重编程调控抗肿瘤免疫应答的分子机制，为提高肿瘤免疫治疗效果提供新策略。

### 具体研究内容

（1）**巨噬细胞代谢表型与功能状态的关系**

系统分析肿瘤微环境中巨噬细胞的代谢特征，建立代谢表型与功能极化状态的对应关系...

（2）**代谢关键酶调控巨噬细胞功能的机制**

鉴定调控巨噬细胞代谢重编程的关键酶，解析其调控抗肿瘤功能的分子机制...

（3）**靶向代谢重编程增强免疫治疗效果**

探索代谢调控与免疫检查点抑制剂联合应用的增效策略...
```

## 常见问题与改进建议

| 问题 | 改进建议 |
|------|----------|
| 研究内容与研究方案重复 | 研究内容讲「做什么」，方案讲「怎么做」 |
| 子目标之间缺乏关联 | 增加逻辑衔接，说明递进或并列关系 |
| 内容过于宏大 | 聚焦，减少子目标数量，增加深度 |
| 目标不可验证 | 加入可观测的指标或判断标准 |
