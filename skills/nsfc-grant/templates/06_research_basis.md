# 研究基础 Prompt 模板

## 系统提示词

```
你正在撰写国自然申请书的「研究基础」部分。

### 核心要求

1. **前期研究积累**
   - 展示与本项目相关的前期工作
   - 说明已有成果和发现
   - 论证研究的延续性

2. **团队构成**
   - 介绍项目负责人和主要成员
   - 说明各成员的专长和分工
   - 体现团队的互补性

3. **实验条件**
   - 列出主要科研平台和设备
   - 说明实验条件的充分性
   - 突出优势资源

4. **合作资源**
   - 介绍重要合作者或单位
   - 说明合作的必要性和互补性

### 格式要求

- 分节组织，层次清晰
- 客观陈述，避免夸大
- 总字数建议：800-1500字
```

## 前期工作展示技巧

### 突出相关性

❌ 罗列所有论文
✅ 只展示与本项目相关的工作

### 强调延续性

```
「在前期工作中，我们发现了[现象]，初步揭示了[机制]，
但[问题]尚不清楚。本项目将在此基础上进一步探索...」
```

### 引用预实验数据

```
「我们的预实验发现，[关键发现]（图X），这为本项目
的假说提供了直接证据。」
```

## 团队组成建议

### 核心成员构成

- **项目负责人**：总体设计、关键指导
- **骨干成员1**：[技术专长]，负责[具体模块]
- **骨干成员2**：[互补专长]，负责[具体模块]
- **研究生**：具体实验执行

### 互补性体现

- 技术互补：分子生物学 + 生物信息学
- 领域互补：基础研究 + 临床转化
- 资源互补：实验平台 + 临床样本

---

## 实验条件展示

### 必备条件

根据研究需要，说明具备以下条件：

| 研究内容 | 所需条件 | 现有情况 |
|----------|----------|----------|
| 细胞实验 | 细胞培养室 | ✓ 已具备 |
| 动物实验 | SPF级动物房 | ✓ 已具备 |
| 流式分析 | 流式细胞仪 | ✓ 已具备 |
| 测序分析 | 测序平台 | ✓ 合作单位 |

## 常见问题与改进建议

| 问题 | 改进建议 |
|------|----------|
| 前期工作与项目关联弱 | 聚焦相关工作，说明衔接 |
| 团队分工不明确 | 用表格清晰列出 |
| 设备堆砌无重点 | 只列与项目相关的关键设备 |
| 缺乏合作说明 | 补充合作基础和分工 |
| 预实验证据不足 | 添加支持假说的初步数据 |
