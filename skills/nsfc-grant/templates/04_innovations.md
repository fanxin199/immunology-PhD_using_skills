# 创新点 Prompt 模板

## 系统提示词

```
你正在撰写国自然申请书的「创新点」部分。

### 核心要求

1. **数量适中**
   - 提炼2-4个创新点
   - 质量重于数量
   - 每个创新点要有实质内容

2. **类型明确**
   - 区分：理论创新、方法创新、应用创新
   - 可混合多种类型
   - 避免类型模糊

3. **具体说明**
   - 明确说明「新在哪里」
   - 与现有研究对比
   - 避免空泛表述

4. **独特性突出**
   - 说明本研究的独特之处
   - 体现项目的核心价值
   - 让评审一眼看出亮点

### 格式要求

- 每个创新点单独成段
- 格式：创新点一：[标题]
- 先概括后展开
- 总字数建议：400-800字
```

## 标准结构模板

```markdown
## 创新点

**创新点一：[标题]**（理论创新）

[首先概括创新点的核心内容（1句话）]。与现有研究相比，本研究[具体说明新在哪里]。这一发现将[说明意义和价值]。

**创新点二：[标题]**（方法创新）

[首先概括创新点的核心内容（1句话）]。传统方法存在[什么问题]，本研究通过[什么策略]解决了这一问题。该方法具有[什么优势]。

**创新点三：[标题]**（应用创新）

[首先概括创新点的核心内容（1句话）]。这为[什么应用场景]提供了[什么新策略/新思路]。
```

## 创新类型详解

### 理论创新

指提出新的科学概念、假说或机制：

- 发现新的分子/信号通路/调控机制
- 提出新的理论框架或模型
- 揭示未知的因果关系

**示例**：
> 创新点一：首次提出「肿瘤相关巨噬细胞代谢-功能偶联」假说，揭示代谢重编程决定TAM极化方向的新机制。

### 方法创新

指开发新的技术方法或研究策略：

- 建立新的实验方法或技术体系
- 创新性地组合已有方法
- 开发新的分析工具或软件

**示例**：
> 创新点二：整合单细胞代谢组学与空间转录组学，建立原位解析免疫细胞代谢状态的新方法。

### 应用创新

指开发新的靶点、疗法或诊断标志物：

- 发现新的治疗靶点
- 开发新的干预策略
- 建立新的诊断/预后标志物

**示例**：
> 创新点三：基于代谢调控策略，开发增强PD-1抗体疗效的联合治疗新方案。

## 常见问题与改进建议

| 问题 | 改进建议 |
|------|----------|
| 过于空泛 | 用具体数据或对比说明 |
| 与现有研究差异不明 | 明确说「与...不同，本研究...」 |
| 创新点太多/太少 | 控制在2-4个，质量胜于数量 |
| 创新类型不明确 | 明确标注理论/方法/应用创新 |
| 表述不够自信 | 避免「可能」「也许」，用肯定语气 |

---

## 避免的表述

❌ 「填补了国内空白」（太空泛）
❌ 「具有重要的理论意义」（没说清楚什么意义）
❌ 「首次研究了...」（首次≠创新）
❌ 「采用了多种先进技术」（方法堆砌不是创新）

## 推荐的表述

✅ 「首次揭示了...的分子机制」
✅ 「与传统...方法不同，本研究...」
✅ 「创新性地将...与...结合」
✅ 「为...提供了新的理论依据/治疗策略」
