# 研究方案 Prompt 模板

## 系统提示词

```
你正在撰写国自然申请书的「研究方案」部分。

### 核心要求

1. **技术路线**
   - 描述清晰的技术路线框架
   - 可用文字描述流程图结构
   - 体现研究的系统性和逻辑性

2. **研究方法**
   - 具体说明每个研究内容采用的方法
   - 方法要具体、可操作
   - 说明方法选择的依据

3. **关键技术与可行性分析**
   - 指出技术难点
   - 说明解决策略
   - 论证方案的可行性

4. **年度进度安排**
   - 分年度安排研究进度
   - 每年有明确的阶段性目标
   - 进度合理、可执行

### 格式要求

- 分节组织，层次清晰
- 技术路线可用流程描述
- 年度安排用表格或列表
- 总字数建议：1500-2500字
```

## 免疫学常用研究方法

### 细胞水平

| 方法 | 应用场景 |
|------|----------|
| 流式细胞术 | 免疫细胞表型、功能分析 |
| 质谱流式 (CyTOF) | 高维度免疫表型分析 |
| 单细胞RNA测序 | 细胞异质性、亚群分析 |
| 空间转录组学 | 组织微环境原位分析 |
| 免疫荧光/免疫组化 | 蛋白定位和表达 |

### 动物模型

| 模型 | 应用场景 |
|------|----------|
| 荷瘤小鼠模型 | 肿瘤免疫研究 |
| 基因敲除/敲入小鼠 | 机制研究 |
| 人源化小鼠 | 人免疫系统研究 |
| 炎症疾病模型（DSS、EAE等） | 炎症免疫研究 |

### 分子水平

| 方法 | 应用场景 |
|------|----------|
| Western Blot | 蛋白表达验证 |
| qPCR | 基因表达检测 |
| ChIP-seq | 转录调控分析 |
| CRISPR筛选 | 功能基因鉴定 |
| 代谢组学 | 代谢物谱分析 |

## 常见问题与改进建议

| 问题 | 改进建议 |
|------|----------|
| 技术路线不清晰 | 用流程图思维，明确输入→方法→输出 |
| 方法描述太笼统 | 具体说明实验条件、参数、步骤 |
| 可行性论证薄弱 | 引用前期工作、文献或预实验结果 |
| 进度安排不合理 | 考虑实验周期，避免前松后紧 |
| 与研究内容脱节 | 每个方法明确对应到研究内容编号 |
