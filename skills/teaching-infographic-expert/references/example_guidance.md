**模块一：抗原提呈细胞 APC**

**1. APC 概念与分类**

**画什么**

- 中间画一枚大 APC，表面有 MHCⅠ/Ⅱ。
- 左边三枚小细胞：DC、巨噬细胞、B 细胞，标“专职性 APC”。
- 右边画上皮细胞、内皮细胞，标“非专职性 APC”。
- 最右再画一枚病毒感染细胞 / 肿瘤细胞，标“靶细胞（广义 APC）”。

**写什么短句**

- APC：加工抗原 → 抗原肽＋MHC → 提呈 T 细胞
- 专职 APC：DC、MΦ、B 细胞
- 非专职 APC：炎症时表达 MHCⅡ
- 靶细胞：MHCⅠ 提呈内源性抗原给 CD8⁺T

**2. 三类专职性 APC 重点功能**

**画什么**

- 并排画三格：DC、巨噬细胞、B 细胞，各自连向 T 细胞。

**写什么短句**

- DC：最强 APC，只能它直接激活初始 T 细胞
- 巨噬细胞：在炎症灶提呈抗原，放大效应 T 细胞功能
- B 细胞：BCR 浓集抗原，向 Th 提呈，获得帮助产生抗体

**模块二：HLA 基因复合体与特点**

**1. HLA 基因区概览（图 5-2）**

**画什么**

- 画一条“6 号染色体短臂”，分成三段：
    - 远端：Ⅰ类区（A、B、C；E、F、G）
    - 中间：Ⅲ类区（补体、TNF 等）
    - 近端：Ⅱ类区（DP、DQ、DR；DM、TAP、PSMB 等）

**写什么短句**

- HLA = MHC，人类位于 6p21.31
- Ⅰ类：A/B/C（经典）＋E/F/G（非经典）
- Ⅱ类：DP/DQ/DR＋DM/TAP/PSMB 等加工基因
- Ⅲ类：补体＋炎症相关基因

**2. HLA 基因特点**

**画什么**

- 画一对父母染色体，分别带不同颜色的 HLA 单倍型，传给子女。
- 细胞膜上画多对不同颜色的 HLA 分子。

**写什么短句**

- 多基因性：Ⅰ类、Ⅱ类各有多基因座
- 多态性：等位基因极多，是移植排斥主要原因
- 共显性：父母等位基因同时表达
- 单倍型遗传：一条染色体上一整套基因一起遗传

**模块三：HLAⅠ/Ⅱ分子结构、分布与功能**

**1. HLAⅠ类分子**

**画什么**

- 单条重链（α1、α2、α3）＋β₂m 组成的分子，插在细胞膜上。
- 远膜端槽两端封闭，槽内放一条“短肽”。
- 旁边标“所有有核细胞”。

**写什么短句**

- 结构：α链＋β₂m，α1/α2 形成肽结合槽
- 肽：8～10 aa，槽两端封闭
- 分布：所有有核细胞
- 功能：提呈内源性抗原肽给 CD8⁺T（CTL）

**2. HLAⅡ类分子**

**画什么**

- 两条跨膜链：α链（α1、α2）＋β链（β1、β2）组成“开口的槽”。
- 槽内放一条“长一点的肽”。
- 旁边标“APC、胸腺上皮、活化 T 细胞”。

**写什么短句**

- 结构：α/β 异二聚体，α1/β1 形成肽结合槽
- 肽：约 13～17 aa，槽两端开放
- 分布：专职 APC、胸腺上皮、活化 T 细胞
- 功能：提呈外源性抗原肽给 CD4⁺T（Th）

**3. “口袋”与“锚定位”**

**画什么**

- 在肽结合槽里画几个凹陷“口袋”，肽上画对应“锚定位”残基。
- 用箭头指明“某些位点配对决定结合稳定性”。

**写什么短句**

- 肽与槽通过“口袋＋锚定残基”精确配对
- 不同 HLA 等位基因 → 不同口袋 → 提呈肽谱不同

**模块四：抗原加工与提呈主干通路**

在画面中下部画一枚 APC，左侧标“内源性抗原 → MHCⅠ途径”，右侧标“外源性抗原 → MHCⅡ途径”。中间用一张小表格列出差异。

**1. 内源性抗原：MHCⅠ途径**

**画什么**

- 左侧画带病毒或突变蛋白的“胞质”。
- 画“泛素标记”的蛋白被送入“免疫蛋白酶体”。
- 由蛋白酶体到 TAP，再到内质网 ER，ER 内有 MHCⅠ＋伴侣蛋白。
- 形成“肽-MHCⅠ复合物”，经高尔基到细胞膜，被 CD8⁺T 的 TCR 识别。

**写什么短句**

- 胞质蛋白 → 泛素化 → 免疫蛋白酶体切割
- 肽经 TAP 进 ER，与 MHCⅠ＋β₂m 装配
- 肽-MHCⅠ 运至膜表，提呈给 CD8⁺T

**2. 外源性抗原：MHCⅡ途径**

**画什么**

- 右侧画细菌 / 蛋白经吞噬或内吞，形成吞噬体 / 内体。
- 与溶酶体融合成酸性囊泡，再与含 MHCⅡ 的囊泡融合成 MⅡC。
- ER 中新合成的 MHCⅡ 与 Ii 链结合，形成（αβIi）₃，经高尔基到 MⅡC，Ii 被降解，仅留 CLIP。
- HLA-DM 去除 CLIP，使抗原肽装载，形成“肽-MHCⅡ”，转运到膜表，被 CD4⁺T 识别。

**写什么短句**

- 外源抗原 → 吞噬/内吞 → MⅡC/溶酶体降解成多肽
- MHCⅡ 在 ER 与 Ii 结合，防止乱装肽
- MⅡC：HLA-DM 促 CLIP 让位，装载抗原肽
- 肽-MHCⅡ 运至膜表，提呈给 CD4⁺T

**3. 小表“Ⅰ类 vs Ⅱ类途径”对照**

**画什么**

- 在两条通路下方画一个 4×2 小表格。

**写什么短句**

- 抗原来源
    - Ⅰ：内源性
    - Ⅱ：外源性
- 主要加工场所
    - Ⅰ：免疫蛋白酶体＋ER
    - Ⅱ：MⅡC / 吞噬溶酶体
- 结合部位
    - Ⅰ：ER 中装配
    - Ⅱ：MⅡC 中装配
- 识别细胞
    - Ⅰ：CD8⁺T（CTL）
    - Ⅱ：CD4⁺T（Th）

**模块五：特殊提呈途径与临床意义**

**1. 交叉提呈（cross presentation）**

**画什么**

- 在 APC 旁边画一个“外源性抗原”箭头，指向既有 MHCⅠ 又有 MHCⅡ 的通路。
- 标注“外源性 → MHCⅠ” 和 “内源性 → MHCⅡ”。

**写什么短句**

- 外源性抗原 → MHCⅠ → CD8⁺T（交叉致敏）
- 内源性抗原 → MHCⅡ → CD4⁺T（少见）
- 对抗病毒、抗肿瘤和诱导耐受很重要

**2. CD1 分子提呈脂类抗原**

**画什么**

- 单独画一条“类 MHCⅠ”的 CD1 分子，槽内放“脂类尾巴”。
- CD1 旁边写 CD1a～e；特别放大 CD1d 与 NKT 细胞相连。

**写什么短句**

- CD1：MHCⅠ样分子，结合脂类抗原
- 提呈给 Th 或 NKT 细胞（CD1d）
- 参与抗分枝杆菌、抗肿瘤与免疫调节

**3. HLA 与临床（移植 + 疾病关联）**

**画什么**

- 画两个小人，一个“供体”，一个“受体”，中间是“HLA 配型”图标。
- 旁边画一个“强直性脊柱炎”脊柱示意，与 HLA-B27 相连。

**写什么短句**

- HLA 匹配度↑ → 器官移植成功率↑
- 特定 HLA 等位基因 ↑ 某些自身免疫病风险（如 B27 与 AS）
