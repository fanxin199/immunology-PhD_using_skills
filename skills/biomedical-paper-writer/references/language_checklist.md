# 学术论文语言规范检查清单

基于 khufkens/paper_writing_checklist。

---

## 语言风格

### ✅ 应该
- [ ] 使用主动语态 ("We analyzed..." 而非 "It was analyzed...")
- [ ] 保持句子简短 (目标: 12-17 词)
- [ ] 使用简洁词汇，避免华丽辞藻
- [ ] 每段落只传达一个核心观点
- [ ] 段落之间逻辑连贯

### ❌ 避免
- [ ] 口语化表达和缩写 (it's, don't, can't)
- [ ] 夸张用语和最高级 (extremely, dramatically, very)
- [ ] 过度使用连接词 (however, moreover, furthermore)
- [ ] 重复同一个词过多次
- [ ] 双重否定

---

## 时态使用

| 章节 | 时态 | 示例 |
|------|------|------|
| Introduction (已知事实) | 现在时 | "Inflammation plays a key role..." |
| Introduction (前人研究) | 过去时 | "Smith et al. reported that..." |
| Methods | 过去时 | "We collected samples..." |
| Results | 过去时 | "Expression increased by 50%..." |
| Discussion (你的发现) | 过去时 | "Our results showed that..." |
| Discussion (普遍真理) | 现在时 | "These findings suggest that..." |

---

## 数字与单位

- [ ] 单独数字 1-9 用文字拼写 (one, two, three)
- [ ] 10 及以上用阿拉伯数字
- [ ] 句首数字用文字拼写
- [ ] 数字与单位之间加空格 (10 mL, 37 °C)
- [ ] 使用标准 SI 单位

---

## 常见问题检查

### 拼写与语法
- [ ] 运行拼写检查
- [ ] 检查主谓一致
- [ ] 检查时态一致性
- [ ] 检查冠词使用 (a/an/the)

### 清晰度
- [ ] 代词指代明确
- [ ] 避免歧义表达
- [ ] 首字母缩略词首次出现时写全称
- [ ] 术语使用一致

### 引用
- [ ] 每个引用都在参考文献列表中
- [ ] 参考文献列表中每条都在正文中引用
- [ ] 引用格式符合目标期刊要求
- [ ] 数据和软件已正确引用

---

## 段落结构检查

每个段落应有：
1. **主题句** - 开头说明本段要点
2. **支撑内容** - 证据、数据、解释
3. **过渡** - 与下段的逻辑连接

---

## 页面排版

- [ ] 12pt 字体 (推荐 Times New Roman 或 Arial)
- [ ] 双倍行距
- [ ] 页码
- [ ] 连续行号
- [ ] 避免段末单独一行 (orphans) 或段首单独一行 (widows)

---

## 提交前最终检查

- [ ] 符合目标期刊的作者指南
- [ ] 所有图表标注清晰
- [ ] 补充材料准备完整
- [ ] 共同作者确认
- [ ] Cover letter 准备完成
- [ ] 利益冲突声明
